# Integrated Speech Analysis System (Speech Insight Pro)

AI 기반 실시간 스피치 분석 및 강사용 관리 시스템입니다.

## 🚀 주요 기능
- **수강생 모드**: 웹캠 녹화, 파일 업로드, AI 정밀 분석 (발음, 속도, 태도), 상세 진단 리포트 제공
- **강사 모드**: 수강생별 성장 그래프, 공통 취약 구간(빅데이터) 분석, 기수별 통계, 학생용 과제(샘플) 실시간 관리
- **실시간 연동**: 수강생의 분석 결과가 강사 대시보드에 즉각 반영

## 🛠 기술 스택
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- Chart.js (성장 그래프)
- Lucide Icons (아이콘)
- LocalStorage / Supabase (데이터 보관)
